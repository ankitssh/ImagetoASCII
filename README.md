# ImagetoASCII
A GUI image to ASCII converter created in C#.

<b>Features</b>

* Drag and Drop feature
* Recent images list
* Click to load recent files
* Well designed GUI
* Supports images from web
* Automatically identifies whether an image is landscape or portrait
